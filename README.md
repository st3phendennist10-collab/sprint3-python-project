# Video Game Sales Analysis

This project contains aggregate data concerning the video game industry and its top 20 highest selling games!

## Data Source
Adapted from VGChartz (public domain estimates).

## Method and Tools
Python

## Key Findings
Finding 1: Wii Sports has the highest global sales, $82.74M.
Finding 2:The sports genre has the highest global sales out of all the genres, $138.45M.
Finding 3: Most of the games are published by Nintendo, 17, 1 by Microsoft and 2 by Take Two.
